在note上添加上vue-router

把note项目改造了一下，采用路由的方式
单页面应用

记录一下

之前的写法

import router from './router'
import Vue from 'vue'

Vue.use(router)

new Vue({
    store, //vuex
    el: '#app',
    router,
    template: '<App/>',
    components: { App } //import App from './App'
})

现在的写法

new Vue({
    store, //vuex
    el: '#app',
    router,
    ...App
})

路由文件 router/index.js

import Router from 'vue-router'
import Index from '@/views/Index'

Vue.use(Router)

export default new Router({
    routes: [
    {
        path: '/index',
        name: 'index',
        component: Index
    },{
        
    }]
})

vue2.0 路由选择 Modes

但需要注意的是，这种情况需要后端的支持，比如locahost:8080/login/:id这种就会找不到

暂时处理方法可以是 定义一个routes * 当所有找不到的path时 都指向404.vue
